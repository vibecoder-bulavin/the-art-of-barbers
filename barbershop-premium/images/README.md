# Снимки за сайта

Временни stock снимки от [Pexels](https://www.pexels.com/) — само мъжка бръснарска тематика (без жени). Заменете със собствени снимки на салона.

## Структура

```
images/
├── hero.jpg          — главна снимка (hero блок)
├── og-cover.jpg      — за социални мрежи / SEO
└── works/
    ├── work-01.jpg   — портфолио 1
    ├── work-02.jpg
    ├── work-03.jpg
    ├── work-04.jpg
    ├── work-05.jpg
    └── work-06.jpg   — портфолио 6
```

## Замяна

Качете файлове с **същите имена**, за да не се променя HTML. Препоръчителен размер: hero ~1200×1500 px, works ~800×800 px (квадрат).
